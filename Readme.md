# Simple C++ Qt4.8 Connect Five Game

![Connect five](/images/connect-five.png) 

### Features
* 2 players
* Network game
* simple Qt gui

## Dependencies

### Run Dependencies for Qt4.8
* libqtgui4
* libqtcore4
* libqt4-network

### Compilation Dependencies on Debian, Ubuntu (for Qt4)
* qt4-qmake
* libqt4-dev
* build-essential

## Compilation
* make

## Run
* 2 players on one PC:
```
./piskvorky
```
